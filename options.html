Textarea OpenAI Rewriter

Hier kannst du deinen OpenAI API-Key lokal in Firefox speichern und das Modell auswählen.

Der Key wird lokal über browser.storage.local gespeichert.
Die Modellliste kann live über die OpenAI API geladen werden. Bildmodelle wie DALL·E und ähnliche Spezialmodelle werden ausgeblendet.

  • Rechtschreibung: Korrigiert nur Rechtschreibung, Grammatik und Zeichensetzung.
  • Optimieren: Verbessert den Text sprachlich.
  • Prompt: Der markierte Text bzw. komplette Feldinhalt wird direkt als Prompt an die API gesendet.
  • Modelle: Beim Auswählen werden kompatible Textmodelle live geladen; DALL·E, Audio-, Embedding- und ähnliche Spezialmodelle werden ausgeblendet.
  • Strg+Z: Nach dem Ersetzen kannst du zum Original zurückspringen.
